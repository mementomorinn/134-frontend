Что такое CSS

CSS (Cascading Style Sheets) — это язык стилей, предназначенный для описания внешнего вида документа, созданного на языке разметки (чаще всего HTML). CSS управляет визуальным оформлением элементов на веб-странице. К его основным функциям относятся:
  • 1. Стилизация
    Определяет, как элементы будут выглядеть (цвета, шрифты, размеры и пр.).
  • 2. Верстка
    Позволяет организовывать элементы на странице (положение и размер).
  • 3. Адаптивность
    Упрощает создание страниц, которые хорошо отображаются на разных устройствах (мобильные, планшеты, ПК).
  • 4. Производительность
    Снижение веса HTML-файлов, с помощью вынесения стилей в отдельные CSS-файлы.

Подключение CSS к HTML-странице

Вы можете подключить CSS к HTML несколькими способами. Рассмотрим два наиболее распространенных:
1. Встроенные стили
Вы можете добавить стили непосредственно в HTML-элементы с помощью атрибута style: 
<p style="color: red;">Это красный текст.</p>
В данном случае, свойство color устанавливает цвет текста. Значение red (красный) определяет, что текст внутри этого параграфа будет отображаться красным цветом.
2. Внутренние стили
Стили можно добавить внутри тега <style> в секции <head> HTML-документа. 
 <head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document</title>
  <style>
    p {
      color: red;
    }
  </style>
</head>
3. Внешние стили
Предыдущие способы могут использоваться в маленьких проектах с небольшим количеством стилей, но что же делать, если ваши стили занимают сто, тысячу и более строк? Для таких случаев существует наиболее удобный и распространенный метод: вы создаете отдельный файл CSS и подключаете его к HTML-документу с помощью тега <link>. Для этого вам нужно добавить стили в файл CSS (в нашем случае файл style.css, расположенный в каталоге CSS), например: 
p {
  color: red;
}
Обратите внимание: каждое правило CSS состоит из свойства и значения, разделенных двоеточием, и заканчивается точкой с запятой. Для группировки стилей, которые относятся к одному селектору используются фигурные скобки. Все правила, описанные между фигурными скобками, будут применяться к элементам, соответствующим этому селектору.
Далее подключите CSS к HTML, прописав в секции <head> файлa index.html следующий код: 
<link rel="stylesheet" href="css/styles.css">
Теперь ваша страница будет использовать стили из файла styles.css, что упростит управление оформлением и выполнит разделение HTML-кода и CSS-стилей.

Классы в CSS

Класс — это атрибут, который можно добавить к HTML-элементу для предоставления ему определенного стиля. Каждый класс начинается с точки (.), после которой идет имя класса. Например, если вы создадите класс с именем highlight, вы будете использовать его в CSS следующим образом: 
.highlight {
  color: red;
}
Классы позволяют группировать элементы и применять к ним одинаковые стили без необходимости дублирования кода. Чтобы применить класс к элементу в HTML, нужно использовать атрибут class. Пример применения класса highlight к элементу <p>выглядит следующим образом: 
<p class="highlight">Это красный текст.</p>
Использование классов способно дать вам много преимуществ, например:
  • 1. Повторное использование
    Один и тот же класс можно применять к многим элементам, что упрощает стилизацию и делает код более лаконичным.
  • 2. Упрощенное управление
    Если вы хотите изменить стиль, вам просто нужно изменить его в одном месте (в файле CSS) вместо того, чтобы менять его в каждом элементе.
  • 3. Комбинирование стилей
    Можно комбинировать несколько классов на одном элементе, чтобы создать уникальный стиль.
  • 4. Чистота кода
    Использование классов помогает поддерживать код более организованным и легко читаемым.
